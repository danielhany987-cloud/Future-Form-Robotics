NOVA

HUMANOID ADAPTIVE LEARNING BIPEDAL

Developed between the ages of 14 and 15, Nova is a custom-built desktop humanoid focused on high-speed computer vision and precise spatial movement. The system utilizes an ESP32 for motor control and an ESP32-CAM for real-time visual data. To achieve fluid motion, I implemented Inverse Kinematics to translate 3D coordinates into servo angles. The AI backend runs on a laptop using OpenCV and NumPy, leveraging a MobileNet-SSD model trained on the COCO dataset for autonomous object detection. The entire mechanical structure was designed and 3D-printed to create a modular, functional research platform.

ARGUS

AI SERVER LOCAL PROCESSING LEGACY HARDWARE

ARGUS is a local AI server designed to run on legacy hardware, enabling powerful machine learning inference without requiring cloud connectivity. This project demonstrates how to optimize neural networks for resource-constrained environments. The system uses quantization and model compression techniques to achieve real-time inference speeds. Built with Python and TensorFlow Lite, ARGUS serves as a backbone for other robotics projects, providing intelligent decision-making capabilities at the edge.

COLOR SENSOR DEVICE

ASSISTIVE TECH ACCESSIBILITY WEARABLE

This wearable color sensor device is designed to assist individuals with colorblindness by identifying colors in real-time and providing audio feedback. Using RGB and spectral sensors, the device analyzes ambient light and classifies colors using a trained neural network. The processed information is communicated to the user via haptic feedback and voice synthesis. This project combines hardware engineering, machine learning, and accessibility design to create a practical tool that improves quality of life.
